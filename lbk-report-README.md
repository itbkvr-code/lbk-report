# Отчёт ЛБК — Воронка «Белая»

Локальное приложение Битрикс24 для аналитики воронки продаж.

## Установка

1. Включите GitHub Pages: Settings → Pages → Deploy from branch `main` → `/ (root)`
2. URL приложения: `https://itbkvr-code.github.io/lbk-report/`
3. В Битрикс24: Разработчикам → Создать приложение → Локальное
4. URL: `https://itbkvr-code.github.io/lbk-report/`
5. Права: `crm`, `user`, `department` (чтение)
6. Установить на портал

## Как работает

- Данные через **Bitrix24 JS SDK** (`BX24.callMethod`) — без вебхуков в коде
- Авторизация через сессию портала
- KPI = реальные totals по стадиям
- Drill-down = последние 200 записей
- Кеш localStorage 5 минут
- Троттлинг 500 мс между запросами

## Безопасность

В репозитории нет токенов и вебхуков. Приложение работает только внутри Битрикс24.
